# Raspberry Pi Touch Display Project

## Description
Ce projet est un système d’interaction IoT simple développé à l’aide d’un Raspberry Pi.
Il permet de détecter une interaction tactile via un capteur tactile capacitif et
d’afficher dynamiquement des messages sur un écran LCD 16x2 en utilisant le protocole I2C.

L’affichage se met à jour automatiquement en fonction de l’état du capteur, ce qui
démontre une programmation événementielle ainsi qu’une interaction entre le matériel
et le logiciel.

## Fonctionnalités
- Détection du toucher à l’aide d’un capteur tactile capacitif
- Affichage dynamique sur un écran LCD (16x2, I2C)
- Anti-rebond logiciel pour éviter les fluctuations indésirables
- Logique basée sur l’état (toucher / relâchement)

## Technologies
- Python
- Raspberry Pi
- GPIO
- I2C
- RPLCD

## Contexte
Ce projet a été développé dans le cadre d’une évaluation pratique en IoT et Raspberry Pi.
Il met en valeur des concepts de programmation fondamentaux applicables au développement
web et mobile, tels que la gestion des événements et des états.
